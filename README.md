## 🛠️ How to Deploy on GitHub Pages

1. **Create a new GitHub repository** named `trackion` (or any name you like)
2. **Upload the files** — drag & drop `index.html` and `README.md` into the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Set branch to `main` and folder to `/ (root)`
6. Click **Save** — your site will be live in ~1 minute

Your site will be available at:
```
https://<your-github-username>.github.io/trackion
```
